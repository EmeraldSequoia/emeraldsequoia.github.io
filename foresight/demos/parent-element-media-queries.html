Static Responsive Image using Media Queries

This demo shows image-set() CSS can work with existing media queries, however this method is really only useful for static images. Dynamically resized images are able fill the exact width of the parent's available width.

The primary content on the left side take 60% of the available width. The content on the right side containing the image takes 40% of the available width.

Adjusting the width of the browser will request various static images.

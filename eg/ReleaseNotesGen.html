Version 3.2_r5931 (2010 Jun 11)

  • Version numbers synchronized with Emerald Chronometer releases
  • Slightly improve reliability of "Disable auto-lock"
  • Change calendar code to be independent of OS changes to calendar database prior to 1582
  • Tap location and NTP status indicators to see details
  • Holding down on windows now accelerates slowly but continuously rather than suddenly going faster
  • Holding down on year windows in Set mode now accelerates to a faster rate for easier access to distant years
  • Help page now linked to developers' blog
  • Add certain small cities whose timezone rules were not represented (e.g., Adak, Alaska and Chatham Islands, NZ)
  • Various minor aesthetic improvements and other bug fixes
